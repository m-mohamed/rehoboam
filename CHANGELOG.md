### Added
- Real-time TUI for Claude Code agent monitoring

- Unix socket IPC for hook events
- Desktop notifications on permission requests
- Multi-terminal support (WezTerm, Kitty, iTerm2)
- Hook installer (`rehoboam init`)
- Tool latency tracking
- Activity sparklines
- GitHub Actions CI
