# Documentation Assets

This directory contains documentation assets for the Rehoboam project.

## Screenshot

To generate `screenshot.png`:

1. Start rehoboam with multiple Claude Code agents in different states
2. Capture the terminal window showing the Kanban dashboard
3. Save as `screenshot.png` in this directory

Recommended dimensions: 800x500 pixels or similar 16:10 aspect ratio.

The screenshot should show:
- At least one agent in each status column (Attention, Working, Compacting, Idle)
- The header with agent counts
- Activity sparklines visible on cards
